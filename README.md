## Getting Started

After cloning the project, run:

```bash
yarn
```

to install the dependencies.

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


To build the project, run:

```bash
yarn build
```
